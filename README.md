# Qiskit Projects

This repository contains my quantum computing experiments and projects built while learning Qiskit.

## Projects

### Bell State Experiment

This project demonstrates:

* Hadamard Gate (H)
* CNOT Gate
* Superposition
* Entanglement
* Measurement

### Result

The Bell State experiment produces approximately:

* 50% |00⟩
* 50% |11⟩

using a quantum simulator.
